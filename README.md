# S A Sanush Portfolio

I developed a modern and interactive personal portfolio website for myself, a Front End Developer based in Kerala, India. The portfolio was created to showcase my technical skills, educational background, projects, resume, and contact details through a visually engaging and fully responsive interface.

I designed the website using modern web technologies with smooth animations, 3D visual elements, and an integrated AI-powered portfolio assistant to provide an immersive and professional user experience. The project highlights my creativity, frontend development skills, and ability to build responsive and interactive web applications.

The portfolio was built with a clean and scalable structure following modern UI/UX principles while ensuring performance optimization, accessibility, and compatibility across different devices and platforms.

## Features

- Responsive personal portfolio design
- Interactive hero section and animated UI elements
- Skills, education, about, and contact sections
- Resume viewing and download support
- Integrated chatbot assistant for portfolio navigation
- GitHub, LinkedIn, and email contact links
- Three.js-powered visual experience

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Three.js
- GitHub Pages
